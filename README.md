## 📦 本地运行

1. 克隆仓库
```bash
git clone https://github.com/yourusername/resignation-calculator.git
```

2. 进入项目目录
```bash
cd resignation-calculator
```

3. 直接用浏览器打开 `index.html` 即可

或者使用本地服务器（推荐）：
```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js (需要先安装 http-server)
npx http-server
```

然后在浏览器访问 `http://localhost:8000`

## 📐 离职指数算法

```
离职指数 = (开会时长 × 10) + (被创次数 × 20) + (续命剂量 × 10) + ((8 - 睡眠时长) × 10)
```

**等级划分：**
- 🔴 **危险** (≥80%)：离职倾向极高，建议立即跑路
- 🟠 **预警** (60-79%)：正在离职边缘，建议骑驴找马
- 🟡 **普通** (40-59%)：还能再苟一苟
- 🟢 **稳定** (<40%)：打工状态尚可
